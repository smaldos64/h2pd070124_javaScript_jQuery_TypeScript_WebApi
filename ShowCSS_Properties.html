På denne side vil forskellige CSS Egenskaber bliver vist !!!

Denne tekst er med blå baggrundsfarve og hvid tekstfarve.

Stylet i CSS som : h2.test { background-color: blue; color: white; }

Og nås i HTML med syntaksen: h2 class="test"



Denne tekst er med grøn baggrundsfarve og hvid tekstfarve.

Stylet i CSS som : h2 .test { background-color: green; color: white; } => læg mærke til mellemrum mellem h2 og .test !!!

Og nås i HTML med syntaksen: h2 div class="test"
